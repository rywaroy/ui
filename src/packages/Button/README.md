## Button

### Button Attributes

| 参数 | 说明 | 类型 | 默认值 |
| - | :-: | - | -:
| type | 按钮类型(primary/default/secondary/normal/warning) | String | default |
| size | 大小(medium / small / mini) | String | medium |
| disabled | 是否禁用 | Boolean | false |
### Button Events

| 事件名称 | 说明 | 回调参数 |
| - | - | - |
| click | 点击事件 | 无
