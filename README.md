# ui

下载依赖

```bash
npm install 
```

启动

```bash
npm run dev
```

打包组件的压缩js文件(在/lib文件夹下)

```bash
npm run build:comp
```

打包组件的样式文件(在/lib/styles文件夹下)

```bash
npm run build:style
```
